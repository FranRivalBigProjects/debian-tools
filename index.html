Debian system monitor tools

The Debian system monitor ecosystem covers CPU, memory, load average, process management, and disk I/O. The default GNOME System Monitor provides a GUI, while CLI tools like htop, btop, and glances offer more detail without a desktop environment.

Network monitoring quick guide

Diagnosing bandwidth issues with iftop and nethogs
Start with sudo iftop -i eth0 to see bandwidth per connection in real-time. The top bar shows aggregate speed. Use arrow keys to sort. Press t to toggle display mode. If you need to know which process is using bandwidth, switch to sudo nethogs eth0 — it shows bandwidth grouped by PID and process name. Press m to toggle between KB/s and cumulative.

Checking open ports with ss vs netstat
ss (from iproute2) is the modern replacement for netstat (from net-tools). Use ss -tuln to list all listening TCP/UDP ports without resolving hostnames. Add -p to show the process name: ss -tulpn. For a specific port: ss -tulpn | grep :80. Install net-tools for netstat: sudo apt install net-tools, then netstat -tuln.

Monitoring CPU and memory in headless Debian servers
For servers without a desktop environment, htop is the standard choice — interactive, colorful, and available in all Debian versions. btop adds graphs and a more modern UI. glances adds network, disk, and sensor data in one view and can serve a web interface on port 61208 with glances -w for remote monitoring. Install all three: sudo apt install htop btop glances.

FAQ — Debian monitoring
For bandwidth per connection: iftop. For bandwidth per process: nethogs. For open ports: ss -tulpn. For network scanning: nmap. For packet capture: tcpdump (CLI) or wireshark (GUI). For routing issues: mtr. Install all at once: sudo apt install iftop nethogs nmap tcpdump mtr bmon.
GNOME System Monitor (gnome-system-monitor) is the default GUI system monitor on Debian GNOME desktops. For servers and headless systems, use CLI alternatives: htop (interactive process viewer), btop (modern with graphs), or glances (all-in-one including network and disk). Install: sudo apt install htop btop glances.
sudo apt install net-tools installs the classic networking tools: netstat, ifconfig, route, arp. Note: net-tools is considered deprecated — the modern replacements are ss (for netstat), ip addr (for ifconfig), and ip route (for route), all from the iproute2 package which is installed by default in Debian.
Quick options: sudo apt install iftop && sudo iftop -i eth0 for real-time bandwidth per connection. sudo apt install nethogs && sudo nethogs for bandwidth per process. sudo apt install bmon && bmon for a bandwidth graph per interface. For cumulative stats: cat /proc/net/dev shows total bytes per interface since boot.
